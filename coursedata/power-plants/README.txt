These files were prepared and kindly shared by Duc Khuat. The folder contains
- A csv file with all fossil fuel based power plants from the global_power_plant_database (https://datasets.wri.org/dataset/globalpowerplantdatabase)
- An R file with all fossil fuel based power plants from the global_power_plant_database 
- A Python script to read in the Power plant coordinates (from the previous list), form a square around it of approximately 50 000km^2 and download the Sentinel-5P data for NO2
